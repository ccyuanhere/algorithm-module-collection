匹配滤波检测测试数据描述
==================================================

数据集大小: 1000
信号长度: 512
信号类型: 实数信号
标签分布: {np.int64(0): np.int64(600), np.int64(1): np.int64(400)}
  - 0: 无目标信号
  - 1: 有目标信号

信号特点:
- 目标信号包含BPSK和QPSK调制
- 挑战性测试数据，包含低SNR和强干扰场景
- 信噪比范围: -2 到 15 dB（包含负SNR）
- 背景噪声强度: 0.1 到 0.8（根据难度调整）
- 噪声类型: 高斯白噪声、有色噪声、干扰信号、相似信号、突发噪声
- 相似信号干扰: 使用部分匹配的序列造成虚警
- 统一使用8位巴克码作为已知序列: [1, 0, 1, 1, 0, 0, 1, 0]
- 对应匹配滤波器模板: [1, -1, 1, 1, -1, -1, 1, -1]

难度分级:
- 简单 (30%): SNR 10-15 dB, 低背景噪声
- 中等 (40%): SNR 3-8 dB, 中等背景噪声
- 困难 (30%): SNR -2-3 dB, 强背景噪声
